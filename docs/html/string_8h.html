Erases part of the string. If start is non-negative, the erased part of the string will start at the start'th position of the src string. If start is negative, the erased part of the string will start at the start'th position from the end of the src string. If len is positive, the erased part of the string will be at most len characters beginning from start (depending on the length of the src string) long. If len is negative, the erased part of the string will contain all characters beginning from start.
