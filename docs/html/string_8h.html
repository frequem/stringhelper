Fetches a portion of the string If start is non-negative, the output string will start at the start'th position of the original string. If start is negative, the substring will start at the start'th position from the end of the original string. If len is positive, the output string will contain at most len characters beginning from start (depending on the length of the input string). If len is negative, the output string will contain all characters beginning from start.
